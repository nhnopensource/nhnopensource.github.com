nhnopensource.org is a space in which we share the open source projects developed by Naver.
Naver continuously contributes to open source projects.

Naver Corporation is Korea's premier Internet company, operating the nation's top search portal, Naver (http://www.naver.com); the leading online game portal, Hangame (http://www.hangame.com;) the nation's largest children's portal, Jr. Naver (http://jr.naver.com); Korea’s the first online donation portal, Happybean (http://happybean.naver.com); and tje microblog service Me2DAY (http://me2day.net/).

Ongoing Projects

  • nGrinder - http://www.nhnopensource.org/ngrinder

    nGrinder is a web-based performance test tool. With nGrinder, you can create stress test scripts, run scripts, monitor target performance and generate reports in a single environment. Base on Grinder (http://grinder.sourceforge.net), nGrinder provides an integrated environment that allows even novice users to create stress tests without a long learning curve.

  • Apache2Nginx - https://github.com/nhnc-nginx/apache2nginx

    NGINX (“engine x”) is a high-performance web server, caching proxy and Layer 7 load balancing solution. Millions of websites on the Internet benefit from using NGINX because of its extreme performance, scalability, reliability, flexibility and security. But like Apache, configuring NGINX is not an easy thing for most people to do. In particular, it takes a lot of effort to learn the modules and directives in Apache and NGINX when we need to migrate to NGINX from an Apache server. For this reason, we developed the apache2nginx tool. The goal of this tool is to generate NGINX configuration file(s) from Apache files.
